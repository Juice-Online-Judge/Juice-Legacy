Judge
=====

## Usage ##
Start server
`rake start`

Stop server
`rake stop`

Open console for model
`rake console`

## Dependency ##
- ruby >= 2.0
- bundler
- rake
- pry

### To Build ###
- scons
- boost >= 1.54

## Build ##
```shell
bundle install
rake build
```
